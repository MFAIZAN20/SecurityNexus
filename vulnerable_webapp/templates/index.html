🎯 About This Application

This is an intentionally vulnerable web application designed for educational purposes to demonstrate common web security vulnerabilities.

🔓 Known Vulnerabilities

  • SQL Injection - Login page vulnerable to SQL injection attacks
  • XSS (Cross-Site Scripting) - Message board allows script injection
  • CSRF - Transfer functionality has no CSRF protection
  • IDOR - Direct object reference in profile URLs
  • Command Injection - Ping tool executes system commands
  • Path Traversal - File download vulnerable to directory traversal
  • Information Disclosure - Search reveals sensitive data
  • Broken Authentication - Weak passwords and session management
  • Missing Authorization - Admin panel accessible without proper checks
  • Insecure Configuration - Debug mode enabled, weak secret key
